Requirements:
- python 3.4
- pip

Application:
- install requiremetns: pip install -r requirements.txt
- create 'logs' folder in the place, where you will run the app
- run app: python app.py (the app will start on port 5000)

Automated tests:
- install requiremetns: pip install -r test-requirements.txt
- run tests: py.test test_app.py --url http://localhost:5000
